1. str (рядки)

Рядки в Python — це послідовність символів, які можна оголошувати в одинарних ' ', подвійних " ".

Для багаторядкових рядків потрійних одинарних ''' ''' або подвійних """ """ лапках.

2. Основні спеціальні символи в рядках

  • Перехід на новий рядок: \n
  • Табуляція: \t
  • Зворотний слеш: \\
  • Гарячі лапки (дві лапки): \"
  • Одинарні лапки: \'
  • Юнікод-символи: \u або \U

5. f-string (форматування рядків у Python 3.6+)

Дозволяє вставляти змінні та вирази безпосередньо в рядок.

      
name = "world"
year = 2025
text = f"Hello, {name}! Year: {year}"
print(text)  # 'Hello, world! Year: 2025'

6. r-string (сирий рядок)

Використовується для обробки спеціальних символів (наприклад, шляхів файлів), не обробляючи зворотні слеші.

      
path = r"C:\Users\Python\Documents"
print(path)        # 'C:\Users\Python\Documents'
print(repr(path))  # 'C:\\Users\\Python\\Documents'

SQL = r'''
SELECT *
FROM tbl
WHERE col = 'value'
'''
print(SQL)
print(repr(SQL))

Виведення SQL-запиту:

      
# Виведення SQL-запиту, як він є, з новими рядками
print(SQL)
